# Welcome

[commitizen-spdx-changelog] is a plugin for [commitizen] that brings YAML
frontmatter awareness to changelog generation — designed for
REUSE/SPDX-compliant projects. No more false version detection from license
headers like `Apache-2.0`.

This documentation follows [Google's Documentation Best Practices].
